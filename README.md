# orientation-api

## Install
```bash
git clone https://github.com/mavidurak/orientation-api.git
cd orientation-api
```
## Setup
```bash
npm install
```
## Set environment variables
Copy `sample.env` file then changes values.
```bash
cp sample.env .env
```
## Migration
Create database then migrate it
```bash
npm run migrate
```
## Start
```bash
npm start
```
## Lint
```bash
npm run lint
```
## Build Hooks
Build git hooks
```bash
npm run build-hooks
```
